### 解決したエラー履歴

| エラー | 原因 | 解決策 |
|-------|------|--------|
| `DependencyHandler.module()` not found | AGP 8.1.0 + Gradle 9.x 非互換 | AGP を 9.1.0 に更新 |
| gradle/wrapper 欠落 | プロジェクト初期作成時に未生成 | gradle-wrapper.jar + properties を手動作成 |
| `Cannot add extension 'kotlin'` | Kotlin プラグイン二重登録 | `app/build.gradle` から `org.jetbrains.kotlin.android` 削除 |
| `kotlinOptions()` not found | AGP 9.x で `kotlinOptions` 廃止 | `compileOptions` のみに統一 |
| `JvmVendorSpec IBM_SEMERU` not found | `jvmToolchain(8)` がデーモン競合を引き起こす | `kotlin { jvmToolchain(8) }` を削除 |

### 1. USBケーブル購入 ← **唯一のブロッカー**
- USB-C ↔ micro USB **データ通信対応**ケーブルが必要
- 充電専用ケーブルでは `adb devices` に表示されないので注意

### 2. ケーブル到着後: Phase 1 疎通テスト

```bash
# ADB 接続確認
adb devices
# 期待: <シリアル番号>    device

# SGP312 確認
adb shell getprop ro.product.model
# 期待: SGP312

# adb forward 設定
adb forward tcp:9000 localabstract:tablet_mirror
adb forward --list

# APK インストール
adb install "/Users/sasata/Documents/Obsidian/タブレットのサブモニタ化/TabletMirrorAndroid/app/build/outputs/apk/debug/app-debug.apk"
```

**動作確認手順**:
1. タブレットで TabletMirror アプリを起動
2. Mac で TabletMirror アプリを起動
3. タイトルバーが「ADB OK」になることを確認
4. メニュー Stream → Send Frame (⌘S) を実行
5. タブレット画面に PNG 画像が表示されれば **Phase 1 完了**

### 3. Phase 2: H.264 ライブストリーミング (ケーブル接続・Phase 1 確認後)

**Mac側** — `VideoEncoder.swift` 実装:
- IOSurface → CVPixelBuffer 変換
- VTCompressionSession (H.264 Baseline, 4Mbps, Bフレームなし)
- SPS/PPS + NAL ユニット Annex B フレーミング
- `StreamSender.swift` の PNG 送信を H.264 送信に差し替え

**Android側** — `MediaCodecReceiver.kt` 実装:
- `OMX.qcom.video.decoder.avc` 初期化
- **同期ポーリングループ必須** (API 19 は非同期モード不可)
- SurfaceView への Surface 出力

## 重要技術メモ

### Android 4.4 (API 19) MediaCodec 制約
```java
// 非同期モード不可 (API 21から) → 同期ポーリング必須
while (running) {
    int inputIndex = codec.dequeueInputBuffer(10000);
    // NAL データを feed...
    int outputIndex = codec.dequeueOutputBuffer(bufferInfo, 10000);
    // Surface に render...
}
```
- H.264 デコーダー: `OMX.qcom.video.decoder.avc`
- `KEY_OPERATING_RATE`, `KEY_PRIORITY` は API 19 に存在しない

### adb forward の仕組み
```
Mac TCP localhost:9000
    ↓ adb forward tcp:9000 localabstract:tablet_mirror
タブレット LocalServerSocket("tablet_mirror")
```
- Android 側は `LocalServerSocket` (abstract socket) — INTERNET 権限不要

### パケットプロトコル
```
┌──────────────────┬──────────────────────┐
│  4 bytes (BE)    │  N bytes             │
│  payload length  │  PNG / H.264 data    │
└──────────────────┴──────────────────────┘
Swift:   UInt32(payload.count).bigEndian
Kotlin:  DataInputStream.readInt() (big-endian 固定)
```
